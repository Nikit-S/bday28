## 📝 Как редактировать

### 1. Текст → `index.html`

Все тексты на цыганскую тематику с романи словами:
- Авлэм = Привет
- Пэрэ мануша = Дорогие люди
- Бэша = Садись
- Пэ = Пей

### 2. Подарки → Google Sheets

Таблица: `name` | `link` | `img` | `booked` | `description`

API в `assets/script.js`.

### 3. Картинки

Замените URL в `index.html` на свои цыганские референсы.
